# new_tab
chrome の new_tab に好きな画像をランダムで表示する拡張機能です.

### 使い方
お好きな画像ファイルを image フォルダ下に格納し, フォルダ内のファイル名から name.csv を生成してください.
Windows 10 でのフォルダ内の全ファイル名が入った csv ファイルを作る方法, および拡張機能の読み込み方は以下を参照してください.
